# Razorpay Utilities Chrome Extension - Feature Guide


## AI Summary

This module helps you add Issue and Resolution notes to tickets automatically using AI, as instructed by GG (Gourab Ghosh).

Features:
- Generates AI-powered Issue and Resolution summary from ticket conversations
- One-click addition to ticket notes
- Custom prompt option to modify the summary format
- API key saved locally (one-time setup)

How to use:
1. For any resolved ticket, click on Add Note
2. Select AI Summary
3. A text box will appear prompting you to enter the Gemini API key

How to get API key:
1. Go to https://aistudio.google.com/
2. Click on Get API key located at the bottom-left corner
3. From the Google AI Studio dashboard, select the default generated key and copy the API key
4. Paste the copied API key into the text box in Freshdesk and click Save

Note: API key only needs to be entered once.


## Daily Tracker

This module helps you track daily ticket work for yourself and your team.

Features:
- Log tickets with status (Resolved, Jira/TP, Transfer/Merge, WOC)
- Quick Add button on ticket pages - one click to add current ticket
- Add comments/notes to each ticket entry
- View team leaderboard - see who resolved most tickets
- Filter tickets by agent
- Select any date to view past entries
- Summary view showing counts by status
- Edit and delete your own entries

How to use:
1. Click Tracker button in Freshdesk header
2. Enter Ticket ID (auto-fills if on a ticket page)
3. Select status
4. Add optional comment
5. Click Add Entry

Quick Add:
- When on a ticket page, click the small arrow next to Tracker button
- Select a status to instantly add the current ticket
